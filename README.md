
# HealthAI Companion - AI Doctor Assistant

A robust, aesthetic React application built with TypeScript and Tailwind CSS. Featuring a dual-interaction model: a landing page with a floating chat widget and a full-featured consultation portal.

## Features
- **Landing Page**: Modern hero section, service cards, and 24/7 floating AI widget.
- **Consultation Portal**: Full-screen chat experience with session history, detailed triage analysis, and AI persona "Rebecca".
- **AI Persona (Rebecca)**: Intelligent mock interaction including triage questions for symptoms like chest pain or fever.
- **Responsive Design**: Mobile-first approach for all pages (Landing, Login, Signup, Chat).
- **Storage**: Persistent data management using LocalStorage.
- **Aesthetics**: Glassmorphism, mesh gradients, and dark mode support.

## Tech Stack
- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Google Material Icons Round
- **State Management**: React Hooks (useChat, useLocalStorage)
- **Data Model**: Structured Conversation and Message interfaces.

## Getting Started
1. Clone the repository.
2. Ensure you have Node.js installed.
3. Run `npm install` to install dependencies.
4. Run `npm run dev` to start the development server.
5. Open your browser and navigate to the application.

*Note: This application uses hash-based routing (#/) to ensure compatibility with single-page deployment environments.*
